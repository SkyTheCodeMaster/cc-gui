slide(win, dir, distance [, delay=0.05])Slides a window in a direction.
slide(win, dir, distance [, delay=0.05])Source

Slides a window in a direction.

Parameters

  1. win table Window to slide.
  2. dir string Direction to slide, has to be "up", "down", "left", or "right".
  3. distance number Distance to slide in pixels. Must be positive.
  4. delay? number = 0.05 Time between repositions.
